# ✈️ Previsão de Preços de Passagens

> Previsão de preços de passagens aéreas com IA. Indica o melhor momento de compra, gera alertas de preço e ajuda o viajante a economizar em cada viagem.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/turismo-precos-passagens/

## 📊 Resultados-chave (KPIs)
**78%** Acerto · **-22%** Economia · **monitor.** Rotas · **auto** Alertas

## 🧠 Tecnologias de IA
ML · Time Series · Price Prediction

## ✨ O que faz
- Previsão de preço por rota
- Melhor momento de compra
- Alertas de queda de preço
- Comparação de companhias
- Economia por viagem

## 🗄️ Fonte de dados real
[ANAC + mercado aéreo](https://www.gov.br/anac) — bases de entrada e resultados disponíveis para download em CSV/Excel no app.
